# BARBER & CO — Сайт барбершопа

Полноценный одностраничный сайт барбершопа с системой онлайн-записи.

## 🚀 Технологии

**Фронтенд:**
- React 18 + Vite 5
- SCSS (BEM-именование)
- Swiper.js 11 (карусель отзывов)
- Flatpickr 4.6 (календарь)
- IMask 7.0 (маска телефона)
- React Router 6

**Бэкенд:**
- Node.js 20 + Express 4.18
- REST API для заявок
- JSON файловое хранилище (легко мигрировать на PostgreSQL)

## 📦 Установка и запуск

### Фронтенд

```bash
cd barber-co
npm install
npm run dev        # Режим разработки
npm run build      # Продакшен-сборка
npm run preview    # Предпросмотр сборки
```

### Бэкенд

```bash
cd barber-co/server
npm install
cp .env.example .env   # Скопируйте и настройте переменные
npm run dev            # Режим разработки с авто-рестартом
npm start              # Продакшен
```

## ✨ Функциональность

### Интерактивные элементы
- **Параллакс-эффект** на Hero-блоке
- **Анимированные счётчики** при прокрутке до секции "О студии"
- **Модальное окно** портфолио с навигацией стрелками и клавиатурой
- **Плавная прокрутка** к секциям при клике на навигацию
- **Валидация форм** в реальном времени с маской телефона
- **Калькулятор стоимости** с мгновенным пересчётом
- **Toast-уведомления** после успешной отправки
- **Переключатель темы** (тёмная/светлая)

### Адаптивность
- **Desktop (1200px+)**: полная версия, сетка 2-3 колонки
- **Tablet (768-1200px)**: 2 колонки, уменьшенные отступы
- **Mobile (до 768px)**: 1 колонка, гамбургер-меню, аккордеон

### API эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/health` | Проверка работоспособности |
| POST | `/api/bookings` | Создание записи |
| GET | `/api/bookings` | Получение всех записей |
| GET | `/api/bookings/:id` | Получение одной записи |
| DELETE | `/api/bookings/:id` | Удаление записи |

Встроена JSON-LD разметка для:
- `LocalBusiness` — адрес, телефон, часы работы, рейтинг
- `Service` — услуги и цены (можно расширить)
- `Review` — отзывы с авторами
